# Data-cleaning-visualization-project
📊 Sales Data Cleaning & Visualization Project

Overview

This project demonstrates a complete data preprocessing and visualization workflow using Python. A raw sales dataset containing missing values, duplicate records, inconsistent formatting, and outliers is cleaned, transformed, and analyzed to generate meaningful business insights.

The project follows industry-standard data cleaning practices and presents findings through visual reports and an automated HTML dashboard.

---

Objectives

- Improve data quality through preprocessing techniques
- Handle missing values, duplicates, and outliers
- Perform exploratory data analysis (EDA)
- Create insightful visualizations
- Generate a professional report for business decision-making

---

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

Data Cleaning Process

The following preprocessing steps were performed:

- Standardized inconsistent text formatting
- Removed duplicate records
- Handled missing values using statistical imputation
- Corrected invalid data entries
- Detected and treated outliers using the IQR method
- Created new analytical features for revenue calculation

---

Visualizations

The project includes:

- Unit Price Distribution Analysis
- Monthly Revenue Trends
- Revenue by Product Category
- Revenue Share by Region
- Correlation Heatmap
- Customer Age vs Revenue Analysis

---

Key Insights

- Identified top-performing product categories
- Determined highest revenue-generating regions
- Analyzed monthly sales performance trends
- Evaluated relationships between customer demographics and revenue
- Improved dataset quality for accurate analysis and reporting
